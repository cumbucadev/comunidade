# Núcleos da Cumbuca Dev — Objetivos e Direcionamento

Este documento define os objetivos de cada núcleo e exemplos de atuação.

Serve como guia para:

* lideranças organizarem melhor o trabalho
* pessoas voluntárias entenderem onde podem contribuir

---

# 📣 Comunicação

## Objetivo

Dar visibilidade à Cumbuca e transformar o que fazemos em conteúdo acessível.

## Atuações principais

* Edição de vídeos (curso, shorts, lives)
* Revisão de legendas
* Produção de conteúdo para redes sociais
* Gestão do WhatsApp (mínimo semanal)
* Informativo mensal
* Comunicação no Slack
* Blog
* Design

## Direcionamento

* Manter consistência de conteúdo
* Criar linha editorial clara
* Melhorar distribuição dos conteúdos

---

# 📚 Curadoria

## Objetivo

Organizar e disponibilizar conteúdos de qualidade para a comunidade.

## Atuações principais

* Curadoria de materiais gratuitos
* Revisão de cursos e conteúdos
* Organização do repositório de curadoria

## Direcionamento

* Criar trilhas de aprendizado
* Classificar conteúdos por nível
* Trabalhar junto com Comunicação

# 🧪 Open Source

* Manutenção de projetos
* Padronização
* Documentação técnica
* Suporte à comunidade

## Rituais

* Office hours mensais de cada projeto Cumbuca Dev
* Sync semanal (sexta 12h–13h)

## Direcionamento

* Criar tarefas para pessoas iniciantes
* Explorar monetização dos projetos

# Como usar este documento

Para lideranças:

* Usar como guia de direção
* Criar tarefas alinhadas com os objetivos

Para pessoas voluntárias:

* Escolher núcleo com mais clareza
* Entender como contribuir

---

# Observação final

Evite criar tarefas sem conexão com o objetivo do núcleo.

Toda atividade deve contribuir para o propósito definido.

Se houverem dúvidas, a coordenação deve sempre ser acionada!
